# Conversor de Monedas Nacional 💱

Aplicación web para convertir pesos chilenos (CLP) a distintas monedas internacionales.  
Diseñada con **HTML, CSS y JavaScript**, optimizada para escritorio y dispositivos móviles.

## 🚀 Demo
Prueba la aplicación en GitHub Pages:  
[https://g1i2s.github.io/currency-converter/](https://g1i2s.github.io/currency-converter/)

## ✨ Características
- Conversión desde CLP a:
  - Dólar (USD)
  - Euro (EUR)
  - UF
  - Bitcoin (BTC)
- Interfaz limpia y responsiva (funciona en móvil y escritorio).
- Estilos modernos con gradiente de fondo y caja central.
- Resultados mostrados de forma clara y centrada.
- Gráfico dinámico en `<canvas>` para visualizar datos.
